Mochi Images

JPG's are great for photos with lots of color without making file size too large but, hey dont support transparent pixels.

GIF's offer simple animation but with limited color palette. NEVER use them for photos.

PNG's are great for anything thats not a photo or animated. They are usually larger file size and have great colors.

SVG's are vector based and can scale to any size without loss of quality. Perfect for responsive design. Use them whenever you can.
